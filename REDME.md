# Fullstack Web Application

A full-stack web application built with **Node.js** for the backend and **Vue 3** for the frontend.  
This project follows a clean structure and is easy to develop, test, and deploy.

---

## Tech Stack

### Backend
- Node.js
- Express
- REST API
- JWT Authentication
- Database: PostgreSQL / MongoDB

### Frontend
- Vue 3
- Vite
- Pinia (state management)
- Axios
